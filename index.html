Projets

Création et configuration d'AD/DS

Configuration d'Active Directory pour gérer un domaine, intégrer des machines et créer des utilisateurs.

Réseaux et protocoles

Connaissances des adresses IP, création de VM sous VirtualBox et ProxMox, mappage de lecteurs réseau.

Création de domaine

Configuration d'un domaine SIO1 avec gestion d'utilisateurs et contrôle des accès via AD/DS.

Configuration de VirtualHost avec Apache2

Installation d'Apache2 et configuration de VirtualHost pour lier des noms de domaine à des adresses IP. Grâce à cette configuration, il est possible d'accéder à un service en utilisant directement le nom de domaine dans le navigateur.

Création d’un environnement virtualisé

Mise en place de machines virtuelles sous Proxmox et VirtualBox pour configurer un Windows Server gérant un Active Directory et un domaine personnel.

Installation et configuration de GLPI

Mise en place et configuration de GLPI pour la gestion des ressources informatiques et le suivi des tickets d'incidents. Ce projet implique l'installation des dépendances nécessaires et la personnalisation de l'outil selon les besoins.
